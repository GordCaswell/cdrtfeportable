Notes:
------

This is cdrtfe 1.5.5

The changes may have introduced new bugs, so please test and report,
if you find a bug.

Some of the translations haven't been updated for quite a while. Any
help regarding the update of existing or the creation of new
translations is much appreciated.


Changes:
--------

* GUI:
  - Support for dpi awareness.
  - Only use system icons in tree- and listviews.
  - Removed old and unused icons.
  - New icons for buttons.
  - Several options which could be set with the cdrtfe.ini are now
    available via the GUI.
              
* DAE:
  - Support for LibParanoia options.

* Disc image:
  - Added 'multisession' option for writing images.

* Activate DEP and ASLR.
                
* Bugfix:
  - Changing the language in portable mode did not save this setting to
    the cdrtfe.ini in the program folder of the portable version. Instead
    the cdrtfe.ini in the user directory was used, if it existed.
  - When no cdrtfe.ini existed, the language setting was saved to the
    cdrtfe_lang.ini. In most cases a user doesn't have the privileges to
    do this. Now a new cdrtfe.ini will be created.
  - Added the cdrtools folder to the search path, so that the code pages
    can be found by mkisofs. 


Hinweise:
---------

Dies ist cdrtfe 1.5.5

Durch die Änderungen könnten neue Fehler entstanden sein. Bitte testen
und Fehler melden.

Einige der Übersetzungen wurden schon eine Weile nicht mehr aktualisiert.
Jedwede Hilfe in bezug auf die Aktualisierung bestehender oder die
Schaffung neuer Übersetzungen ist sehr willkommen.


Änderungen:
-----------

* GUI:
  - Anpassungen für 'dpi awareness'.
  - Ausschließlich System-Icons in Tree- und Listviews verwenden.
  - Alte, zusätzliche Icons entfernt.
  - Neue Icons für Buttons.
  - Verschiedene Einstellungen, die bereits über die cdrtfe.ini 
    gesteuert werden konnten, auch über das GUI verfügbar machen.
              
* DAE:
  - Unterstützung für die LibParanoia-Optionen.

* Disk-Image
  - Option 'Multisession' hinzugefügt.

* DEP und ASLR aktivieren.
                
* Bugfix:
  - Wenn im Portable-Mode die Spracheinstellungen geändert werden und 
    eine cdrtfe.ini im User-Verzeichnis existiert, wurde diese Änderung
    in diese Datei gespeichert anstatt in der cdrtfe.ini der portablen
    Version.
  - Wenn Spracheinstellungen geändert wurden, ohne daß eine cdrtfe.ini
    existiert, wurden diese Änderungen in die cdrtfe_lang.ini geschrieben,
    wofür die nötigen Rechte in der Regel nicht vorhanden sind. 
    Stattdessen wird jetzt eine neue cdrtfe.ini angelegt.
  - cdrtools-Ordner in den Suchpfad aufnehmen, damit mkisofs die Zeichen-
    satztabellen finden kann. Bei den Downloads (Installer und Archive)
    die Ordnerstruktur korrigiert (lib\siconv statt siconv).
